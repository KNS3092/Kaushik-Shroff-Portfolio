Graph Neural Network Accelerator

Project Overview: This project involved designing a pipelined Graph Neural Network (GNN) accelerator for parallel task scheduling across heterogeneous Arm A53/A57 cores. The architecture was implemented in System Verilog, with key functional units including MAC, ReLU, and Aggregation modules. Emphasis was placed on timing optimization, area efficiency, and power reduction through bit-width pruning, modular pipelining, and sign-bit-aware ReLU logic. The full frontend-to-backend flow was executed using ModelSim, Design Compiler, Innovus, PrimeTime and Virtuoso with a 7nm standard-cell library. Post-layout simulation and analysis yielded a maximum frequency of 869 MHz, power of 3.07 mW, and an energy-delay-area product (EDAP) of 7.91 pJ·ns·mm². Although clock gating reduced power by 38% pre-APR and 5% post-APR, it was rolled back post-APR due to timing violations—highlighting key trade-offs in design-space exploration.

Key Takeaways:

  • Designed a modular GNN accelerator using Verilog with MAC, ReLU, and Aggregation stages for multi-task scheduling.
  • Achieved 869 MHz post-layout frequency and 3.07 mW power on a 7nm node with Design Compiler and Innovus.
  • Optimized energy efficiency with a measured EDAP of 7.91 pJ·ns·mm² using bit-width pruning and RTL-level enhancements.
  • Implemented clock gating for 38% dynamic power savings pre-APR and 5% post-APR.
  • Completed full frontend-to-backend RTL-to-GDSII flow, including post-layout power/timing analysis with PrimeTime.
