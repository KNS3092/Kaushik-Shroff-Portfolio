Scan Based Test Generation

Project Overview: This project focused on generating full and partial scan-based ATPG solutions for the ITC’99 Viper (b14) processor using Synopsys tools. Full scan synthesis was performed in Design Vision, followed by ATPG execution in TetraMax, achieving a test coverage of 99.48%. Partial scan designs were created using SCOAP-based flip-flop ranking to prioritize scan candidates based on controllability and observability. A total of 17 design configurations were analyzed for trade-offs in area, test time, and coverage. Test efficiency was benchmarked using the M metric (TC/A·L·N·P), with one partial scan configuration showing a 122% improvement over full scan. The entire process was automated with custom Tcl scripts for scan chain generation, metric extraction, and test compression flow.

Key Takeaways:

  • Achieved 99.48% test coverage on the b14 Viper processor using full scan and TetraMax ATPG patterns.
  • Designed partial scan architectures with SCOAP-based ranking to optimize test coverage and resource utilization.
  • Evaluated 17 scan configurations, with up to 122% improvement in test efficiency.
  • Automated scan chain generation, ATPG flow, and coverage reporting using custom Tcl scripting.
  • Explored trade-offs in scan coverage, area overhead, pattern count, and fault detection time across multiple designs.
